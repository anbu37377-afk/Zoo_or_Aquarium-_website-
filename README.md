# City Zoo & Aquarium Website

A modern, responsive website for a City Zoo & Aquarium, featuring timed ticketing, attraction highlights, and a dark/light theme.

## Features
- **Responsive Design**: Built with Bootstrap 5.
- **Dark/Light Mode**: Toggle between themes for a personalized experience.
- **Interactive UI**: Hover effects and smooth transitions.
- **Pages Included**:
  - Home
  - About Us
  - Services/Attractions
  - Pricing/Tickets
  - Blog
  - Contact
  - User Dashboard
  - Login/Register

## Hosting
This website is hosted using GitHub Pages.

## Technologies Used
- HTML5
- CSS3 (Custom Styles + Bootstrap 5)
- JavaScript (Vanilla)
- Bootstrap Icons
